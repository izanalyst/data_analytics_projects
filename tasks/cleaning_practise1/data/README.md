# Data Cleaning Project

## 📊 Dataset
Concert tours dataset (artists, revenue, shows)

## 🎯 Goal
Clean raw data and prepare it for analysis

## 🛠 Tools
- Python
- Pandas

## 🧹 Cleaning Steps
- Removed reference markers ([1], [2], etc.)
- Cleaned monetary values ($, commas)
- Converted columns to numeric types
- Processed year column into start and end year

## 📈 Result
Data is cleaned and ready for analysis
